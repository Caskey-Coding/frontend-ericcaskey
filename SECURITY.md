# Security Policy

This repository is part of the Caskey Coding platform. Our full security
policy, data retention policy, and privacy practices are maintained in:

https://github.com/Caskey-Coding/caskeycoding-specs/tree/main/_shared/security

## Reporting a vulnerability

If you discover a security vulnerability in this code, please email
**e@ericcaskey.com** with the subject line **"Security: frontend-ericcaskey"**.

We aim to acknowledge reports within 72 hours and remediate confirmed
vulnerabilities within:

- 7 days for high / critical severity
- 30 days for medium severity
- 90 days for low severity

Please do not file public GitHub issues for security reports.
