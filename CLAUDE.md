# Repository notes for Claude

## Prose style rules

- **No em-dashes (`—`) anywhere in site copy, metadata, commit messages, or comments.** Rewrite with a comma, semicolon, colon, parentheses, or a new sentence. This applies to every `.tsx`, `.md`, and `.ts` file in this repo, and to any content pasted in from drafts.
- Prefer plain ASCII punctuation in user-visible strings. Curly quotes, en-dashes, and other typographic characters are allowed only when they are part of a proper name or a JSON-LD field that requires them.

When editing any file in this repo, search for `—` (U+2014) before committing and replace every occurrence.
